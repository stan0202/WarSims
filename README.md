# WarSims: 聖焰與暗影的交鋒

WarSims 是一個基於純前端技術 (HTML5 / CSS3 / Vanilla JavaScript) 開發的 **JRPG 風格 3x3 九宮格自動戰鬥模擬器**。
無需任何後端伺服器，直接透過瀏覽器開啟 `index.html` 即可遊玩，並且完全支援 GitHub Pages 部署。

## 🌟 遊戲特色

*   **直覺的拖曳佈陣系統**：透過 HTML5 Drag and Drop 技術，玩家可以輕鬆將角色卡片拖曳至九宮格中，或是直接在盤面上調整陣型。
*   **深度的屬性克制與戰鬥邏輯**：
    *   **戰士** ➔ 剋制 ➔ **刺客** ➔ 剋制 ➔ **弓手** ➔ 剋制 ➔ **戰士** (克制時傷害 1.5 倍並觸發爆擊音效)。
    *   攻擊目標邏輯：根據角色職業特性（偏好前排或後排）智慧鎖定敵人，帶來多變的戰局。
*   **極致的視聽體驗**：
    *   **玻璃擬態 UI**：充滿現代感與奇幻質感的半透明介面與高解析度排版。
    *   **CSS 動畫**：支援衝刺攻擊、受擊震動、浮動傷害跳字與陣亡漸隱效果。
    *   **無縫音效引擎**：利用 Audio CloneNode 技術，支援高頻率攻擊時的音效重疊，並具備音量控制與靜音切換功能。
*   **資料大數據匯出**：結合瀏覽器 `LocalStorage` 技術，每一次的對局佈陣與勝負結果都會記錄在本機，玩家可隨時點擊「匯出戰鬥紀錄」下載 JSON 檔案供未來分析使用。

## 🎮 如何遊玩

1.  **環境準備**：若您希望有完整的音效體驗，請將四首對應的 MP3 放入遊戲根目錄（`bgm.mp3`、`hit.mp3`、`crit.mp3`、`die.mp3`）。
2.  **開啟遊戲**：使用現代瀏覽器開啟 `index.html` 即可。
3.  **選擇人數與陣容**：決定雙方上場人數 (2~5 人)，透過拖曳角色卡片或點擊「🎲 一鍵隨機佈陣」來完成雙方九宮格佈署。
4.  **開始戰鬥**：點擊「⚔️ 開始戰鬥」，系統便會根據單雙數回合輪替先手，自動演繹完整場戰鬥。

## 📁 專案架構

*   `index.html`: 遊戲主頁面與 UI 骨架。
*   `style.css`: 所有的視覺樣式、動畫效果與 RWD 設計。
*   `js/`: 核心遊戲模組目錄，包含 UI 互動 (`app.js`)、音效模組 (`audio.js`)、對戰引擎 (`battle.js`) 與角色設定 (`characters.js`)。
*   `scripts/`: Python 模擬與輔助腳本，包含早期開發版本留下的命令行模擬器 (`simulate.py` 及其變體)。

## 🚀 部署至 GitHub Pages

由於本專案為純靜態檔案，部署極為容易：
1.  將整個專案推送到 GitHub Repository。
2.  進入 GitHub Repository 的 **Settings** > **Pages**。
3.  將 Source 設為 `Deploy from a branch`，並選擇您的主分支 (如 `main`) 後按下 Save。
4.  等待幾分鐘後，您的遊戲就可以分享給全世界遊玩了！
